# Assignment Submission Reminder Agent

## Question

Build an agent that tracks assignment deadlines and reminds students about upcoming submissions, adjusting reminders based on individual tendencies.

## Requirements

1. Collect assignment deadlines and student-specific procrastination patterns.
2. Model each student's usual submission timing and urgency level.
3. Decide on the timing and frequency of reminders.
4. Adapt reminders based on the student's response and prior behavior.
